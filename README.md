# 🌐 Site Inovata
Landing page institucional para a empresa **Inovata**, desenvolvida com HTML, CSS e JavaScript.  
Projeto simples e responsivo, com foco em identidade visual e apresentação de serviços.

## 📌 Sobre o projeto
Este site estático foi criado para representar a presença online da Inovata, com as seguintes seções:
- **Cabeçalho com logo e navegação**
- **Seção institucional ("Sobre nós")**
- **Serviços oferecidos**
- **Formulário de contato**
- **Rodapé com redes sociais**

Design responsivo para mobile, tablet e desktop.

## 🖥️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (puro)
- Bootstrap
